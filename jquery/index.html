Variables

What are variable types, declarations and scope?


Variable is like a bowl that can contain a certain value. The bowl itself does not do anything, but it takes a role as a means to keep a data value. The way to keep data is called ‘Declaration’, and it is declared with the equals sign and a given name(identifier).

Depending on where the variable is declared, it can be ‘Block’ type when it is set within a block with ‘let’, or ‘Local’ when it is declared within a function, otherwise the scope is ‘Global’, out of a function. However, in a ‘Global’ case, it should have a reason to use as a whole page.

Overall, variable simplifies complex calculations to avoid writing long and complex statements multiple times.
There are 3 types: var, let and const. ‘var’ is a general type of variable and the same ‘var’ can be used multiple times, in this case the last value will be stored. ‘let’ is usually used locally within a block statement and the same ‘let’ can’t be used multiple times. The read-only variable is ‘const’.
const x = 1 ;
var y = 2 ;
// Here x is 1, y is 2
   {
   let y = 3 ;
   // Here x is 1, y is 3
   }
// Here x is 1, y is 2

Conditions

What are the conditional structures in JS and how might you use them?


When we need to make a decision, conditional structure is used under the certain condition. A simple approach is to use a conditional operator, also known as a ternary operator as it takes three operands. However, it is more common to use if statement for complex calculations. This is very useful in many cases as it calculates the number of different cases and outputs the appropriate value. For example, it returns grades based on student performances, it returns the amount due based on the items in the shopping cart.

There are 3 different ways to express: ‘If statement’, ‘If else statement’ and ‘else if statement’. ‘If statement’ does not care of a single condition whereas ‘If else statement’ considers when this condition is both true and false. As for ’else if statement’, 1 or more conditions are nested to initial statement.
If time is less than 10:00, create a "Good morning" greeting, if not, but time is less than 20:00, create a "Good day" greeting, otherwise a "Good evening":
if  (time <  10 ) {
   greeting =  "Good morning" ;
}  else if  (time <  20 ) {
   greeting =  "Good day" ;
}  else  {
   greeting =  "Good evening" ;
}
