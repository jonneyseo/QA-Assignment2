jQuery


Selector syntax and types of selectors


The way jQuery approaches to DOM is as same as CSS, they use selector. A basic jQuery selector syntax looks like

$(selector).action();
$ is the prefix that identities where jQuery begins.

Selector can be #id, .class, div or this. Action can be a DOM attribute like text, CSS attribute, event handler or function. The best advantage to use jQuery instead of JavaScript is it shortens codes, which enables us to quickly access to DOM. For example, to select id ‘apple’, jQuery expresses $(‘#apple’), whereas JavaScript has to use document.getElementById(‘apple’).
The jQuery element selector selects elements based on the element name.
$(document). ready( function (){
  $( "button" ).click( function (){
    $( "#test" ).hide();
  });
});
// When a user clicks a button, the element with id="test" will be hidden.

jQuery


Some examples of DOM get and set actions


We can ‘get’ and ‘set’ from DOM. In this sentence, As I interpret, ‘get’ means ‘read-only’ and ‘set’ is on ‘change’ something. For example, DOM ‘get’ can be calls and prints some texts that are already written in a class. DOM ‘set’ can print certain texts directly without bringing them from somewhere.
$( "button" ).click( function (){
  $( "h1, h2, p" ).addClass( "blue" );
  $( "div" ).addClass( "important" );
});
//Classes are added to the elements
$( "button" ).click( function (){
  $( "h1, h2, p" ).removeClass( "blue" );
});
//Class is removed from the elements

jQuery


Some example of effects or animation actions


jQuery has useful built-in effects including Hide and Show, Fade In and Out, Slide Up and Down. These are literally doing as its name suggest. For Animate effect, it performs a set of CSS properties such as moving left or right, changing colours and sizes.
$( "button" ).click( function (){
  $( "#div1" ).fadeIn();
  $( "#div1" ).fadeIn( "slow" );
  $( "#div3" ).fadeIn( 3000 );
});
//Three targets fade in in different ways
$( "button" ).click( function (){
  $( "#div" ).animate({left: '250px' });
});
//The target moves 250px to the right
