Process

Scrolling: The project required to show only one content at a time for a total of three content. There are many ways, but I chose to scroll. To do so, I accommodated a jquery solution called ‘pagepilling’.

Details: Since the purpose of the work was to show a basic understanding of JavaScript with examples, I wanted to create a code pen (the dedicated space within a div tag) where the actual HTML coding environment is simulated.

Chart JS: When I watch the news or YouTube, I often see failures to properly visualize information. The current status of COVID-19 was visualized using Chart.js. To get across better to the audience, the graph type, colour, legend, animation were customized.
